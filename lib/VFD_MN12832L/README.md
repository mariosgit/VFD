# Usage

### Within PlatformIO

You can either...
* clone this repo into yout lib folder
* same but as submodule ```git submodule add <repo addr> lib/VFD_MN12832L```
* add repo to platformio.ini ```lib_deps = mariosgit/VFD_MN12832L```
